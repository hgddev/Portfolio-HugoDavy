À propos du projet

Le but de ce projet était de créer une chaîne de traitement automatisée pour gérer efficacement les fichiers d'une potentielle infrastructure de création de site web.

Ces scripts permettent de convertir des images dans le format demandé par le client, de creer des rapports pdf à partir de tableurs, ainsi que de trier les fichiers dans des dossiers output spécifiques en fonction de leur type.


Un rapport de logs est ensuite créé afin de résumer les opérations qui ont été éffécuées, et de signaler les éventuelles erreurs rencontrées lors du processus de traitement.

01.

Algorithmie

Optimisation des boucles de traitement pour réduire le temps d'exécution sur des fichiers lourds.

02.

Fiabilité

Mise en place d'un système de "Check-sum" pour vérifier que les fichiers n'ont pas été altérés.
